In the JavaScript language, describe all the use cases for each of the following:

parenthese ()

Parentheses are used to call a function, and to enclose the parameters of a function. They can also used to determine order of execution - just like in basic arithmic. For example (2x3) + (2x3) = 36.

brackets []

Square brackets [] are used to create, hold, define and retrieve elements from an array. For example use var newArray = []. Then elements can be added easily to that array. Once created they are used to access the elements in the array. Square brackets [] can be used to nest arrays inside another array - to create multi-dimensional arrays. For example [[1,2,3] [4,5,6]]

braces {}

Braces are used to enclose statement when writing a function so that you know where the function begins and ends. Another use is to enclose all the properties of an object when defining it in the form of object literal notation. These braces {} are also used to group multiple statements together e.g. if multiple statements are required to be run in a loop. Don't underestimate their importance.

single quotes '' and double quote ""

Single and double quotes can be used interchangeably and are used for holding strings. A string can be any text inside double or single quotes. For example var myString = "text inside quotes". String indexes are zero-based - the first character is in position 0, the second in 1, and so on. It can be useful to use both single and double quotes to nest strings inside other strings. For example var myString = "The string said 'hello world' erday."
