What is Scope and how does it work in JavaScript?

In JavaScript scope is the set of variables that objects and functions you have access to. Scope changes depending on where the varaibale is mentioned or 'called'.

Variables delcared within a javascript function have LOCAL scope, and are only relevant to that function. Since local variables are only recognised inside their functions, variables with the same name can be used in different functions - without any confusion.

A variable declared outside a function, becomes GLOBAL. Global variables have a global scope meaning all scripts and functions on the page can access and use it.
