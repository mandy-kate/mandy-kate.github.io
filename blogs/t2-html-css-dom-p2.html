What is grid based design?

Grid based design takes advantage of the way people read (left - right, then top - bottom). It uses columns and rows within the page to make the most out of a page layout. A grid is split in rows and columns and HTML elements are placed in each area of the grid. Typically a row is divided into 12 columns (1/12 = 8.33% of the page, 2/12 = 16.66% of the page and so on). 12 is a good number to use because the width of a page is typically divided into 2,3 or 4 sections and 12 is the lowest number that is divisible by all those numbers.

Whats all the hype about responsive webpages?

People use many different devices to access the internet. Each device is different - not just in the size of screen but also in terms of the resolution of the screen (pixels). Responsive design means your website looks great for everyone regardless of the device they are using. When implementing responsive design the main thing you need to remember is not to use fixed pixel size to determine width - but instead state the pixel as a percentage of the screen. For example, 300 pixels = 24% of the page (this is an estimate, and depends on the screen being used).

What is semantic structure?

Semantic HTML is the use of HTML to reinforce the meaning (semantics) of the information in webpages and web applications rather than merely to define its presentation or look.
